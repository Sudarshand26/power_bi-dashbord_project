# power_bi-dashbord_project #
= Blinkit Sales Analysis Dashboard

## Project Overview ##
This project analyzes Blinkit sales data using Power BI. 
The dashboard provides insights into sales performance, product categories, and customer trends.

## Tools Used ##
- Power BI
- Excel / Dataset
- Data Cleaning
- Data Visualization

## Key Insights ##
- Total Sales Analysis
- Category-wise Sales
- Outlet Performance
- Item Type Analysis
- Customer Rating Insights

## Dashboard Features
✔ Interactive Filters  
✔ Sales KPIs  
✔ Category Breakdown  
✔ Outlet Size Analysis
